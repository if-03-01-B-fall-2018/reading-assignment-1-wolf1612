2. Identify void elements and how they are denoted

HTML elements with no content are called empty elements. Empty elements can be "closed" in the opening tag like this: < br/> or this < p/>. HTML5 does not require empty elements to be closed. But if you want stricter validation, or if you need to make your document readable by XML parses, you must close all HTML elements properly.

The empty elements in HTML are: < area > < base > < br > < col > < embed > < hr > < img > < < input > < keygen > < link > < meta > < param > < source > < wbr >

3. Identify attributes

The href Attribute (HTML links are defined with the < a > tag. The link address is specified in the href attribute: < a href="https://www.sucuk.com">This is a link< /a>

The src Attribute (HTML images are defined < img > tag.The filename of the image source is specified in the src attribute: < img src ="hello_world.jpg" >)

The width and height Attributes (Images in HTML have a lot of size attributes, which specifies the width and height of the image: < img src ="hello_world.jpg" width="470" height="600" >)

The alt Attribute (The alt attribute specifies an alternative text to be used, when an image cannot be displayed. The value of the attribute can be read by screen readers. This way, someone "listening" to the webpage, e.g. a blind person, can "hear" the element) < img src="hello_world.jpg" alt="Program which prints hello world" >

The style Attribute (The style Attribute is used to specify the styling of an element, like color, font,size etc.) < p style="color:green">Hello< /p>

The lang Attribute (The lang attribute specifies the language of the element's content. Common examples are "en" for English, "es" for Spanish, "fr" for French, and so on.) < html lang="en-US">

The title Attribute (The value of the title attribute in < p > will be displayed as a tooltip when you mouse over the paragraph) < p title="I'm a loottip">
